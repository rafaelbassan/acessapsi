
```
Mobile:     1 coluna

```

## 7. Acessibilidade

### 7.1 Requisitos WCAG 2.1
- Contraste mínimo 4.5:1 para texto normal
- Contraste mínimo 3:1 para texto grande
- Navegação por teclado
- Screen reader compatibility
- Focus indicators visíveis

### 7.2 Semantic HTML
```html
<main>
  <section aria-labelledby="professionals-heading">
    <h2 id="professionals-heading">Nossos Profissionais</h2>
    <article>
      <h3>Dr. João Silva</h3>
      <p>Especialista em...</p>
      <button aria-label="Agendar consulta com Dr. João Silva">
        Agendar
      </button>
    </article>
  </section>
</main>
```

## 8. Micro-interações

### 8.1 Hover Effects
- Botões: Subtle scale (1.02x) + shadow
- Cards: Lift effect (shadow elevation)
- Links: Color transition

### 8.2 Loading Animations
- Skeleton loading para cards
- Spinner para ações
- Progress bars para uploads

### 8.3 Success Feedback
- Toast notifications
- Check mark animations
- Color transitions

Esta especificação de UI/UX garante uma experiência consistente e profissional em todas as plataformas do PsiClin.
